# Guia rapido

```bash
yarn create next-app [project-name] -e https://github.com/odiegoalessandro/nextjs-with-styled-components
# or
npx create-next-app [project-name] -e https://github.com/odiegoalessandro/nextjs-with-styled-components
```

Após isso configure suas fontes no `src/lib/globalStyles.ts`.
